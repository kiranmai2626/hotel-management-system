# Hotel Management System

This is a mini project demonstrating a hotel room booking system.

## Features
- View rooms
- Book room
- Booking confirmation
- Cancel booking

## Technologies Used
- Java (Console Application)
- HTML, CSS, JavaScript

## How to Run
### Java
- Compile: javac HotelManagement.java
- Run: java HotelManagement

### Web
- Open index.html in browser
